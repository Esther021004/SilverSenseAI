# 🚀 실행 가이드 (터미널 문제 해결)

터미널이 멈췄거나 문제가 있을 때 사용하는 가이드입니다.

## ⚡ 빠른 실행 방법

### 방법 1: 명령 프롬프트(CMD)에서 직접 실행

1. **Windows 키 + R** 눌러서 실행창 열기
2. `cmd` 입력하고 Enter
3. 다음 명령어로 프로젝트 폴더로 이동:
   ```
   cd "C:\generative Ai project"
   ```

4. **패키지 확인 및 설치**:
   ```
   python check_packages.py
   ```
   → 누락된 패키지가 있으면 안내된 명령어로 설치

5. **벡터 스토어 구축**:
   ```
   python build_vectorstore.py
   ```
   → 5-15분 정도 소요됩니다 (첫 실행 시)

6. **검색 테스트**:
   ```
   python test_search_only.py
   ```

---

### 방법 2: 배치 파일 사용 (Windows)

1. **`RUN_ME.bat`** 파일을 더블클릭
   → 자동으로 벡터 스토어 구축 시작

2. 구축이 완료되면 **`TEST_SEARCH.bat`** 더블클릭
   → 검색 기능 테스트

---

### 방법 3: PowerShell에서 실행

1. 프로젝트 폴더에서 **Shift + 우클릭** → "PowerShell 창 여기에서 열기"

2. 명령어 실행:
   ```powershell
   python check_packages.py
   python build_vectorstore.py
   python test_search_only.py
   ```

---

## 🔧 문제 해결

### Q: "python을 찾을 수 없습니다" 오류
**해결책**: Python이 PATH에 없는 경우

1. Python 설치 경로 확인 (예: `C:\Users\사용자명\AppData\Local\Programs\Python\Python310\`)
2. 전체 경로로 실행:
   ```
   C:\Users\사용자명\AppData\Local\Programs\Python\Python310\python.exe build_vectorstore.py
   ```

또는 Python Launcher 사용:
```
py build_vectorstore.py
```

---

### Q: 패키지 설치 오류
**해결책**: 필수 패키지만 먼저 설치

```bash
pip install pypdf pdfplumber langchain langchain-community chromadb sentence-transformers python-dotenv
```

---

### Q: 메모리 부족 오류
**해결책**: 
- 다른 프로그램 종료
- 문서를 나눠서 처리 (나중에 구현 가능)

---

### Q: 임베딩 모델 다운로드가 너무 느림
**해결책**: 
- 인터넷 연결 확인
- 처음 한 번만 다운로드되므로 기다려주세요

---

## 📝 단계별 체크리스트

- [ ] Python 설치 확인: `python --version` 또는 `py --version`
- [ ] 프로젝트 폴더로 이동: `cd "C:\generative Ai project"`
- [ ] 패키지 확인: `python check_packages.py`
- [ ] 벡터 스토어 구축: `python build_vectorstore.py` (5-15분)
- [ ] 검색 테스트: `python test_search_only.py`

---

## 💡 팁

1. **첫 실행은 시간이 걸립니다**
   - 임베딩 모델 다운로드: 5-10분
   - 문서 처리: 2-5분
   - 총 5-15분 정도 소요

2. **두 번째부터는 빠릅니다**
   - 벡터 스토어가 이미 있으면 자동으로 로딩
   - 모델도 이미 다운로드되어 있음

3. **진행 상황 확인**
   - 터미널에 진행 상황이 출력됩니다
   - "✅" 표시가 나오면 완료된 단계입니다

---

## 🆘 그래도 안 되면?

1. **에러 메시지 확인**: 터미널에 나온 오류 메시지 전체를 복사
2. **파일 확인**: 
   - `document/` 폴더에 PDF 파일이 있는지
   - `requirements.txt` 파일이 있는지
3. **Python 재설치**: Python 3.8 이상 버전 확인

---

## ⚙️ 대안: Python 스크립트로 직접 실행

터미널이 계속 문제가 있으면, Python IDLE나 VS Code에서 직접 스크립트를 열어서 실행할 수도 있습니다:

1. `build_vectorstore.py` 파일 열기
2. F5 키를 눌러 실행 (또는 Run 버튼)

단, 이 경우 터미널 출력이 바로 보이지 않을 수 있습니다.
